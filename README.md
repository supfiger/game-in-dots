# Game in dots
Test task for "Starnavi" company using React, lodash.

### Watch demo:
https://supfiger.github.io/starnavi-game-in-dots/

### To run and view the project just run next commands:
  - **`yarn install`**
  - **`yarn start`**
  - And open in browser [localhost:3000](http://localhost:3000/)
